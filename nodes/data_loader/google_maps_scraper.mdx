### 노드 입력

- `query`: Google Maps에서 찾고자 하는 장소나 정보를 나타내는 검색어입니다. 예를 들어, 특정 카페 이름이나 위치에 대한 키워드를 사용할 수 있습니다.
- `language`: 검색 결과의 언어를 지정합니다. 기본값은 "en"(영어)이며, "ko"(한국어) 등 다른 언어도 설정할 수 있습니다.
- `latitude`: (선택사항) 검색의 기준이 되는 위도입니다. 특정 위치 주변에서 결과를 필터링할 때 사용합니다.
- `longitude`: (선택사항) 검색의 기준이 되는 경도입니다. 특정 위치 주변에서 결과를 필터링할 때 사용합니다.
- `radius`: (선택사항) 검색 반경을 미터 단위로 설정합니다. 설정된 반경 내에서 검색 결과를 제한합니다.
- `max_results`: 가져올 최대 검색 결과 수를 설정합니다. 기본값은 10이며, 필요한 경우 더 많은 결과를 요청할 수 있습니다.

### 노드 출력

- `name`: 각 결과의 장소 이름 리스트입니다.
- `description`: 각 결과에 대한 간단한 설명입니다. 장소의 특징이나 주요 정보를 포함합니다.
- `address`: 각 장소의 주소 리스트입니다.
- `thumbnail`: 각 장소의 썸네일 이미지 링크입니다.
- `category`: 각 장소의 분류나 카테고리 정보입니다. 예: 카페, 레스토랑 등
- `latitude`: 각 장소의 위도 정보 리스트입니다.
- `longitude`: 각 장소의 경도 정보 리스트입니다.
- `reviews`: 각 장소에 대한 리뷰 수입니다.
- `google_maps_link`: 각 장소의 Google Maps 링크입니다.
- `website`: 각 장소의 웹사이트 링크입니다.
- `phone`: 각 장소의 전화번호입니다.

### 기능

GoogleMapsScraper 노드는 주어진 검색어와 위치 정보를 바탕으로 Google Maps에서 장소 정보를 스크랩하여, 해당 장소의 이름, 설명, 주소 등의 다양한 정보를 제공합니다. 특정 위치나 반경 내에서 장소를 필터링하거나 특정 언어로 결과를 표시할 수 있으며, 최대 결과 수를 조절하여 원하는 양의 정보를 수집할 수 있습니다.

### 언제 사용할까요?

이 노드는 특정 지역이나 키워드에 맞는 장소 정보를 자동으로 수집하고자 할 때 유용합니다. 예를 들어, 특정 위치 근처의 카페, 레스토랑, 명소 등의 정보를 수집하여 사용자에게 제공하거나, 위치 기반의 서비스에서 장소 데이터를 검색하는 데 활용할 수 있습니다. 특히, 장소에 대한 구체적인 정보가 필요한 경우 또는 여러 장소에 대해 일괄적으로 데이터를 수집해야 할 때 효과적입니다.
