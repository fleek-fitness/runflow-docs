### 노드 입력

- `url`: YouTube 동영상의 URL입니다. 다운로드할 대상 URL을 지정합니다.

### 노드 출력

- `video`: 다운로드된 동영상 파일을 나타냅니다. 파일은 `FileType` 형식으로 반환되며, S3 URL을 포함합니다.

### 기능

YoutubeDownloader 노드는 주어진 YouTube URL을 사용하여 동영상을 다운로드하고, 다운로드된 파일을 AWS S3 버킷에 업로드합니다. 동영상 파일이 성공적으로 업로드되면 S3 URL이 반환되어 다른 노드에서 사용할 수 있도록 합니다. 이 노드는 단일 URL 또는 여러 URL에 대한 동영상 다운로드를 지원합니다.

### 언제 사용할까요?

이 노드는 YouTube에서 동영상을 직접 다운로드하여 AWS S3에 저장하고자 할 때 유용합니다. 예를 들어, 특정 YouTube 콘텐츠를 자동화된 워크플로우에서 다운로드하고 저장하여 이후 분석 또는 공유할 때 사용할 수 있습니다. 대량의 동영상 파일을 수집하거나 백업 목적으로 사용할 수 있습니다.
