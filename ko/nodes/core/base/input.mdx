### 노드 입력

- `value`: 사용자로부터 입력받는 값입니다. 이 값은 노드의 출력으로 그대로 전달됩니다.

### 노드 출력

- `output`: 입력된 `value`와 동일한 값이 출력됩니다.

### 기능

InputNode는 워크플로우에서 사용자가 제공한 특정한 값을 단순히 전달하는 노드입니다. 이 노드는 주어진 `value`를 출력으로 전달함으로써 다른 노드로 데이터를 전송하는 역할을 수행합니다.

### 언제 사용할까요?

이 노드는 사용자가 지정한 값이나 설정을 다른 노드로 전달하고자 할 때 유용합니다. 예를 들어, 워크플로우 초기 단계에서 특정한 값을 설정하고 그 값을 이후의 노드에서 사용할 때 활용할 수 있습니다. 간단한 값의 전달을 필요로 할 때 사용하기 적합합니다.
