---
- `input1` (선택적, string 또는 string[]): 텍스트의 첫 번째 부분을 입력합니다. 문자열 또는 문자열 리스트 중 하나로 입력할 수 있습니다.
- `input2` (선택적, string 또는 string[]): 텍스트의 두 번째 부분을 입력합니다. 문자열 또는 문자열 리스트 중 하나로 입력할 수 있습니다.

### 노드 출력

- `combinedText` (string 또는 string[]): 입력된 텍스트를 주어진 형식에 맞춰 결합한 결과입니다. 단일 문자열 또는 여러 문자열 리스트로 반환됩니다.

### 기능

텍스트 결합기 노드는 두 개 이상의 텍스트 입력을 받아 사용자 정의 형식에 맞게 결합된 텍스트를 만듭니다. 입력값은 지정된 형식의 위치에 삽입되며, 특정 입력이 없을 경우 해당 부분은 건너뜁니다.

### 언제 사용할까요?

텍스트 결합기 노드는 아래 상황에서 유용합니다:

- 여러 정보를 합쳐서 개인 맞춤형 메시지를 만들 때
- 데이터를 특정 형식으로 맞춰 보고서나 출력물을 작성할 때
- 자동화된 워크플로우에서 동적 텍스트 생성 및 구조화가 필요할 때
