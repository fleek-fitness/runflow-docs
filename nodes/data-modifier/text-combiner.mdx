### 노드 입력

- `input1` (선택적, string 또는 string[]): 결합할 텍스트의 첫 번째 부분입니다. 문자열 또는 문자열 리스트 형식으로 입력할 수 있습니다.
- `input2` (선택적, string 또는 string[]): 결합할 텍스트의 두 번째 부분입니다. 문자열 또는 문자열 리스트 형식으로 입력할 수 있습니다.

### 노드 출력

- `combinedText` (string 또는 string[]): 입력된 텍스트가 지정된 형식에 맞게 결합된 결과 텍스트입니다. 단일 문자열 또는 여러 문자열 리스트 형식으로 반환됩니다.

### 기능

TextCombiner 노드는 두 개 이상의 텍스트 입력을 받아 사용자가 정의한 형식에 따라 결합된 텍스트를 생성합니다. 입력된 값은 지정된 형식에서 정의한 위치에 삽입되며, 특정 입력이 주어지지 않을 경우 해당 부분은 생략됩니다.

### 언제 사용할까요?

TextCombiner 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 여러 정보를 결합하여 사용자 맞춤 메시지 생성
- 데이터를 특정 형식에 맞춰 보고서 또는 출력물 생성
- 동적인 텍스트 생성이 필요한 자동화 워크플로우에서 텍스트 결합 및 구조화
