### 노드 입력

- `file` (FileType 또는 FileType[]): 자동 크롭할 비디오 파일을 지정합니다. 비디오 파일의 URL을 포함하는 `FileType` 객체를 단일 항목 또는 리스트로 제공할 수 있습니다.
- `aspect_ratio` (string): 원하는 크롭 비율을 설정합니다. 지원되는 비율은 `1:1`, `4:3`, `3:4`, `4:5`, `5:4`, `9:16`이며, 기본값은 `9:16`입니다.
- `active_speaker_detection` (boolean): 활성 화자 감지 기능을 사용하여 주요 화자에 집중하여 비디오를 크롭할지 여부를 설정합니다. 기본값은 `True`입니다.

### 노드 출력

- `cropped_video` (FileType[]): 설정된 크롭 비율과 활성 화자 감지 설정을 기반으로 크롭된 비디오 파일입니다. 각 비디오는 URL과 파일 이름이 포함된 `FileType` 객체로 제공됩니다.

### 기능

VideoAutocropper 노드는 입력된 비디오를 특정 화면 비율로 자동으로 크롭하는 기능을 제공합니다. 또한, 활성 화자 감지 기능을 통해 주요 화자가 프레임에 잘 맞도록 조정합니다. 크롭된 비디오는 지정된 비율에 맞춰 새롭게 생성되며, 작업의 효율성을 높이기 위해 비디오 파일들을 병렬로 처리할 수 있습니다.

### 언제 사용할까요?

VideoAutocropper 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 소셜 미디어에 맞는 세로형 또는 특정 비율의 비디오 클립이 필요한 경우
- 회의 녹화 또는 프레젠테이션 영상에서 주요 화자에 집중된 크롭 비디오가 필요할 때
- 다양한 비디오 소스를 빠르게 자동 크롭하여 여러 포맷의 콘텐츠를 생성할 때

VideoAutocropper는 비디오 자동화 워크플로에서 중요한 장면을 포착하여 특정 비율로 빠르게 편집할 때 유용한 도구입니다.
