### 노드 입력

입력 노드는 흐름의 시작점으로, 다양한 소스로부터 데이터를 받을 수 있게 해줍니다.

#### 노드 출력

- `input` (어떠한 유형): 입력 노드에서 받아들인 데이터로, 소스에 따라 다양한 유형일 수 있습니다.

### 노드 기능

입력 노드는 흐름 내 데이터의 진입점 역할을 합니다. 여러 소스로부터 정보를 받을 수 있습니다. 예를 들어:

- 사용자 입력
- API 호출
- 데이터베이스 쿼리
- 파일 업로드
- 웹훅 페이로드
- 예약된 이벤트

받아들인 데이터는 이후 흐름의 다른 노드에서 사용될 수 있습니다.

### 사용 시기

입력 노드를 사용해야 할 때:

- 외부 데이터를 사용해 새 흐름을 시작해야 할 때
- 특정 이벤트나 조건을 기준으로 흐름을 시작해야 할 때
- 사용자 입력이나 API 요청을 받아야 할 때
- 워크플로우로 외부 데이터를 가져와야 할 때

### 예시

입력 노드를 사용하는 몇 가지 예시입니다:

1. 채팅 인터페이스에서 사용자 입력 받기
2. 자동화된 워크플로우용 웹훅 페이로드 수락
3. 흐름에서 처리할 CSV 파일의 데이터 가져오기
4. 예약된 이벤트에 따라 흐름 시작하기

<Info>
  입력 노드의 구체적인 구성 옵션은 사용하는 플랫폼과 작업 중인 입력 유형에 따라 달라질 수 있습니다.
</Info>
