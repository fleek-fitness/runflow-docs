### 노드 입력

- `file` (FileType 또는 FileType[]): 자막을 생성할 비디오 파일입니다. `FileType` 객체로 제공되며 단일 항목 또는 리스트로 입력할 수 있습니다.
- `template_id` (string): 자막 생성에 사용할 템플릿 ID입니다. 기본값은 `07ffd4b8-4e1a-4ee3-8921-d58802953bcd`입니다.
- `language` (string): 자막 언어를 지정합니다. 기본값은 영어(`en`)입니다.
- `auto_approve` (boolean): 생성된 자막을 자동으로 승인할지 여부를 설정합니다. 기본값은 `True`입니다.

### 노드 출력

- `video` (FileType): 자막이 추가된 비디오 파일입니다. URL과 파일 이름이 포함된 `FileType` 객체로 제공됩니다.
- `video_url` (string): 자막이 포함된 비디오의 다운로드 URL입니다.

### 기능

VideoCaptionGenerator 노드는 입력된 비디오 파일에 템플릿 기반의 자막을 자동으로 추가하는 기능을 제공합니다. 사용자는 템플릿 ID와 언어 설정을 통해 자막 스타일과 언어를 선택할 수 있습니다. 자막이 생성되면 비디오 파일에 자막이 적용되어 저장되며, 생성된 비디오의 URL을 출력합니다.

### 언제 사용할까요?

VideoCaptionGenerator 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 비디오 콘텐츠에 자막을 자동으로 추가하여 접근성을 높이고자 할 때
- 다양한 언어로 자막을 추가해 다국어 지원 콘텐츠를 제작할 때
- 일정한 템플릿을 사용하여 일관된 스타일의 자막을 생성하고자 할 때

VideoCaptionGenerator는 비디오 자막 자동화를 위한 간편한 워크플로 노드로, 비디오 편집 시간과 노력을 절약할 수 있습니다.

- `aspect_ratio` (string): 원하는 크롭 비율을 설정합니다. 지원되는 비율은 `1:1`, `4:3`, `3:4`, `4:5`, `5:4`, `9:16`이며, 기본값은 `9:16`입니다.
- `active_speaker_detection` (boolean): 활성 화자 감지 기능을 사용하여 주요 화자에 집중하여 비디오를 크롭할지 여부를 설정합니다. 기본값은 `True`입니다.

### 노드 출력

- `cropped_video` (FileType[]): 설정된 크롭 비율과 활성 화자 감지 설정을 기반으로 크롭된 비디오 파일입니다. 각 비디오는 URL과 파일 이름이 포함된 `FileType` 객체로 제공됩니다.

### 기능

VideoAutocropper 노드는 입력된 비디오를 특정 화면 비율로 자동으로 크롭하는 기능을 제공합니다. 또한, 활성 화자 감지 기능을 통해 주요 화자가 프레임에 잘 맞도록 조정합니다. 크롭된 비디오는 지정된 비율에 맞춰 새롭게 생성되며, 작업의 효율성을 높이기 위해 비디오 파일들을 병렬로 처리할 수 있습니다.

### 언제 사용할까요?

VideoAutocropper 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 소셜 미디어에 맞는 세로형 또는 특정 비율의 비디오 클립이 필요한 경우
- 회의 녹화 또는 프레젠테이션 영상에서 주요 화자에 집중된 크롭 비디오가 필요할 때
- 다양한 비디오 소스를 빠르게 자동 크롭하여 여러 포맷의 콘텐츠를 생성할 때

VideoAutocropper는 비디오 자동화 워크플로에서 중요한 장면을 포착하여 특정 비율로 빠르게 편집할 때 유용한 도구입니다.

VideoHighlightGenerator 노드는 입력된 비디오에서 특정 기준(prompt)에 맞는 하이라이트를 찾아 클립을 생성하는 기능을 제공합니다. 예를 들어, "most viral moments"와 같은 설정으로 비디오를 분석하여 가장 주목받을 만한 순간들을 하이라이트로 추출하고, 결과를 비디오 클립 형태로 제공합니다. 사용자는 `render_clips` 옵션을 통해 클립을 실제 비디오로 렌더링할 수 있습니다.

- 긴 비디오에서 특정 주제에 맞는 주요 장면을 자동으로 추출하고 싶을 때
