### 노드 입력

- `output` (string 또는 string[]): 전달할 단일 문자열 또는 문자열 목록입니다. Node의 입력 값으로 제공됩니다.

### 노드 출력

- `output` (string 또는 string[]): 입력으로 받은 `output` 값을 그대로 반환합니다.

### 기능

OutputNode는 주어진 `output` 값을 그대로 전달하여 워크플로우의 끝부분에서 데이터를 출력하는 역할을 수행합니다. 입력된 값을 다른 시스템이나 프로세스로 내보내기 위한 마지막 단계의 Node입니다.

### 언제 사용할까요?

OutputNode는 다음과 같은 상황에서 유용하게 사용됩니다:

- 워크플로우의 종료 단계에서 데이터를 출력할 때
- 최종 결과를 내보내거나 다음 프로세스로 전달해야 할 때

<Info>
  OutputNode는 결과를 단순히 전달하기 때문에 데이터 변경이나 변환을 수행하지
  않습니다.
</Info>
