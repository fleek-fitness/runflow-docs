---
title: "AI 번역기"
description: "한 언어에서 다른 언어로 텍스트를 번역합니다."
---

### 노드 입력

- `content` (string 또는 string[]): 번역할 텍스트입니다. 문자열 또는 문자열 리스트 형식으로 입력할 수 있습니다.
- `targetLanguage` (string): 번역할 대상 언어의 코드입니다. 예를 들어, 영어는 'en', 한국어는 'ko'로 입력합니다.

### 노드 출력

- `text` (string 또는 string[]): 번역된 텍스트입니다. 입력된 `content`를 `targetLanguage`로 변환한 결과가 반환됩니다. 단일 텍스트 또는 여러 텍스트로 반환될 수 있습니다.

### 기능

Translator 노드는 주어진 텍스트를 사용자가 지정한 언어로 번역합니다. 사용자는 번역할 텍스트(`content`)와 목표 언어(`targetLanguage`)를 제공하고, 노드는 해당 언어로 정확하게 번역된 텍스트를 반환합니다. 비동기적으로 동작하여 여러 텍스트를 동시에 처리할 수 있습니다.

### 언제 사용할까요?

Translator 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 다국어 지원이 필요한 워크플로우
- 사용자가 여러 언어로 제공된 정보를 특정 언어로 변환해야 할 때
- 다국어 번역 작업을 자동화할 때
- 번역 서비스가 필요한 모든 시나리오

### 예시

1. 고객 지원 챗봇에서 사용자의 메시지를 서비스 언어로 번역
2. 다국어 뉴스 피드에서 특정 언어로 기사 번역
3. 다국어 제품 설명을 단일 언어로 통합

<Info>
  정확한 번역을 위해 신뢰할 수 있는 번역 API 또는 서비스와 연동하여 사용하는
  것이 좋습니다.
</Info>
