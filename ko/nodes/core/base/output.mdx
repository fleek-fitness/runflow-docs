### 노드 입력
- `output`: 노드가 받아들이는 입력값입니다. 이 값은 `OutputNode`가 전달받아 최종 출력으로 제공됩니다.

### 노드 출력
- `output`: 입력된 `output` 값을 그대로 반환합니다.

### 기능
OutputNode는 워크플로우에서 다른 노드의 결과나 데이터를 최종적으로 수집하여 출력하는 역할을 합니다. 이 노드는 워크플로우의 마지막 단계에서 데이터를 모으고, 해당 값을 결과로 출력합니다.

### 언제 사용할까요?
이 노드는 워크플로우의 마지막 단계에서 최종 결과를 확인하거나 저장하고자 할 때 사용됩니다. 예를 들어, 여러 노드에서 처리된 데이터를 최종 결과로 출력하거나 사용자에게 데이터를 전달할 때 적합합니다. 입력받은 값을 그대로 반환하여 최종 출력을 정의할 때 유용합니다.
