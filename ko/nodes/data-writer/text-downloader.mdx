### 노드 입력

- 파일 이름`(문자열): 저장할 파일의 이름입니다.
- 텍스트` (문자열): 파일에 저장할 텍스트입니다.

### 노드 출력

이 노드는 출력을 생성하지 않지만 입력 텍스트를 지정된 파일명으로 로컬 파일에 저장하는 작업을 수행합니다.

### 기능

TextDownloader 노드는 사용자가 제공한 텍스트를 지정된 파일명을 사용하여 로컬 파일에 저장합니다. 기본 파일 형식은 '.txt'이며, 사용자는 파일명과 텍스트 내용을 모두 입력으로 제공합니다. 이 노드를 사용하면 텍스트 데이터를 파일에 쉽게 저장하고 보존할 수 있습니다.

### 언제 사용하나요?

TextDownloader 노드는 다음과 같은 상황에서 유용하게 사용할 수 있습니다:

- 프로세스 중에 생성된 텍스트 데이터를 로컬 파일에 저장해야 하는 경우.
- 로그, 보고서 또는 기타 텍스트 정보를 파일에 보존하려는 경우.
- 자동화된 워크플로우에 텍스트 저장 단계를 포함해야 하는 경우.

<Info>이 노드는 입력 텍스트를 로컬 파일에 저장하여 데이터를 쉽게 관리하고 보존할 수 있도록 도와줍니다.</Info>
