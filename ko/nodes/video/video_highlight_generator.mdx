### 노드 입력

- `file` (FileType 또는 FileType[]): 분석할 비디오 파일을 지정합니다. 비디오 파일의 URL을 포함하는 `FileType` 객체를 단일 항목 또는 리스트로 제공할 수 있습니다.
- `prompt` (string, 선택 사항): 비디오 하이라이트 생성 기준을 정의하는 텍스트입니다. 기본값은 "most viral moments"이며, 분석할 하이라이트의 테마를 설정할 수 있습니다.

### 노드 출력

- `videos` (FileType[]): 생성된 하이라이트 비디오 파일 목록입니다. 각 비디오는 URL과 파일 이름이 포함된 `FileType` 객체로 구성됩니다.
- `highlight_urls` (string[]): 하이라이트 비디오의 URL 리스트입니다.
- `title` (string[]): 생성된 하이라이트의 제목 리스트입니다.
- `score` (number[]): 각 하이라이트의 중요도를 나타내는 점수 리스트입니다.
- `start_time` (number[]): 각 하이라이트 시작 지점(초) 리스트입니다.
- `end_time` (number[]): 각 하이라이트 종료 지점(초) 리스트입니다.
- `duration` (number[]): 각 하이라이트의 길이(초) 리스트입니다.

### 기능

VideoHighlightGenerator 노드는 입력된 비디오에서 특정 기준(prompt)에 맞는 하이라이트를 찾아 클립을 생성하는 기능을 제공합니다. 예를 들어, "most viral moments"와 같은 설정으로 비디오를 분석하여 가장 주목받을 만한 순간들을 하이라이트로 추출하고, 결과를 비디오 클립 형태로 제공합니다. 사용자는 `render_clips` 옵션을 통해 클립을 실제 비디오로 렌더링할 수 있습니다.

### 언제 사용할까요?

VideoHighlightGenerator 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 긴 비디오에서 특정 주제에 맞는 주요 장면을 자동으로 추출하고 싶을 때
- 소셜 미디어에 적합한 짧은 하이라이트 클립을 생성할 때
- 대량의 비디오 파일에서 빠르게 중요한 장면을 분석하고 하이라이트를 얻고자 할 때

이 노드는 특히 콘텐츠 제작 워크플로에서 비디오 요약이나 주요 장면 추출이 필요할 때 유용하게 사용할 수 있습니다.
