---

### 노드 입력

- `content` (string 또는 string[]): 문장을 나눌 텍스트 또는 텍스트 리스트입니다. 문자열 또는 문자열 리스트 형식으로 입력할 수 있습니다.

### 노드 출력

- `sentences` (string[] 또는 string[][]): 입력된 `content`를 문장 단위로 분리한 결과입니다. 하나의 텍스트가 입력되면 문장 리스트로 반환되며, 여러 텍스트가 입력되면 각 텍스트에 대한 문장 리스트의 리스트로 반환됩니다.

### 기능

SentenceSplitter 노드는 입력된 텍스트를 문장 단위로 분리하는 역할을 합니다. 주어진 텍스트에 대해 문장 부호를 기준으로 문장을 나누며, 여러 개의 텍스트를 동시에 처리할 수 있습니다. 각 문장은 구분자로 사용된 공백이나 기호 등을 제거하고 깔끔하게 반환됩니다.

### 언제 사용할까요?

SentenceSplitter 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 긴 텍스트를 여러 문장으로 나누어 분석하거나 처리해야 할 때
- 자연어 처리 워크플로우에서 문장 단위로 데이터를 세분화해야 할 때
- 대화형 AI 시스템에서 문장을 분리하여 개별적인 작업을 수행할 때

### 예시

1. 긴 기사나 문서를 여러 문장으로 나누어 텍스트 분석
2. 사용자의 입력을 문장 단위로 분리하여 챗봇에서 처리
3. 여러 텍스트에서 각각의 문장을 리스트로 출력하여 자연어 처리

<Info>
  문장 단위로 텍스트를 세분화하여 다양한 NLP 작업에 활용할 수 있습니다.
</Info>
