### 노드 입력

- `file` (file 또는 file[]): 처리할 파일 또는 파일 리스트입니다. 파일 유형에 따라 다른 처리가 적용됩니다.
- `text` (string 또는 string[]): 처리할 텍스트 또는 텍스트 리스트입니다. 텍스트 데이터에서 정보를 추출할 때 사용됩니다.

### 노드 출력

- 추출된 데이터 (object 또는 object[]): 설정된 `schemas`에 따라 파일이나 텍스트에서 추출된 정보입니다. 각 키는 스키마에 정의된 대로 추출된 값을 포함하며, 단일 객체 또는 객체 리스트로 반환될 수 있습니다.

### 기능

DataExtractor 노드는 파일이나 텍스트에서 구조화된 데이터를 추출하는 기능을 제공합니다. 스키마에 정의된 형식에 맞춰 데이터를 분석하고 필요한 정보를 반환합니다. PDF 형식의 파일이나 이미지 파일을 적절히 처리하며, 텍스트 데이터에서도 정해진 스키마에 따라 정보를 추출합니다.

### 언제 사용할까요?

DataExtractor 노드는 다음과 같은 상황에서 유용합니다:

- 문서에서 특정 키워드, 설명, 등록번호 등을 추출해야 할 때
- 자동화된 데이터 수집이나 문서 분석이 필요할 때
- 대규모 데이터에서 특정 정보를 찾아야 할 때
