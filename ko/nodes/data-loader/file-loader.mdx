### 노드 입력

- `file` (file 또는 file[]): 로드할 파일 또는 파일 리스트입니다.

### 노드 출력

- `file` (file 또는 file[]): 성공적으로 로드된 파일 또는 파일 리스트입니다.

### 기능

FileLoader 노드는 지정된 파일을 다운로드하여 로컬 시스템에 저장합니다. 입력으로 제공된 파일의 URL을 사용하여 파일을 가져오고, 로컬 경로에 저장하는 작업을 자동으로 수행합니다. 이 노드는 단일 파일 또는 여러 파일을 동시에 처리할 수 있으며, 설정된 동시성 제한 내에서 병렬로 파일 다운로드를 실행합니다.

### 언제 사용할까요?

FileLoader 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 원격 서버에서 파일을 다운로드해야 할 때
- 워크플로우에서 외부 리소스를 불러와야 할 때
- 대량의 파일을 자동으로 처리해야 할 때
- 여러 파일을 동시에 처리하거나 대규모 파일 불러오기 작업을 자동화할 때

### 예시

1. 외부 API에서 제공하는 리소스를 주기적으로 다운로드하여 로컬 시스템에 저장
2. 여러 서버에서 분산된 데이터를 수집해 로컬 환경에 통합
3. 대규모 파일을 워크플로우의 특정 노드로 넘기기 전 로드 작업 자동화

<Info>
  FileLoader를 사용할 때, 네트워크 상태와 서버 응답 시간에 따라 작업이 지연될 수 있으니 이를 고려하여 워크플로우를 설계해야 합니다.
</Info>
