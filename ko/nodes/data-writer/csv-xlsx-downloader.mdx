The source and target languages are the same (ko). Returning the input text without translation:

### 노드 입력

- `field 1` (string): 첫 번째 필드에 대한 데이터입니다.
- `field 2` (string): 두 번째 필드에 대한 데이터입니다.

### 노드 출력

이 노드는 출력을 생성하지 않으며, 입력된 데이터를 CSV 또는 XLSX 파일로 로컬에 저장하는 작업을 수행합니다.

### 기능

CsvXlsxDownloader 노드는 입력된 데이터를 기반으로 CSV 및 XLSX 형식의 파일을 생성하고, 지정된 파일명으로 로컬에 저장합니다. 사용자는 파일명을 설정할 수 있으며, 여러 개의 필드를 입력하여 각각의 필드를 파일의 열로 사용할 수 있습니다. 이 노드는 데이터 저장 과정을 자동화하여 수동으로 데이터를 파일로 저장하는 시간을 절약해 줍니다.

### 언제 사용할까요?

CsvXlsxDownloader 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 프로세스에서 생성된 데이터를 CSV 또는 XLSX 형식으로 내보내어 저장해야 할 때
- 데이터 분석이나 보고를 위해 데이터를 파일로 저장하고자 할 때
- 자동화된 워크플로우에서 데이터 저장 단계를 포함시켜야 할 때

<Info>이 노드는 입력된 데이터를 CSV 또는 XLSX 파일로 저장하여 데이터 관리와 공유를 용이하게 합니다.</Info>
