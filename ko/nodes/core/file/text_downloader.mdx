### 노드 입력

- `File name` (string): 저장할 파일의 이름입니다.
- `Text` (string): 파일에 저장할 텍스트입니다.

### 노드 출력

이 노드는 출력을 생성하지 않으며, 입력된 텍스트를 지정된 파일명으로 로컬에 저장하는 작업을 수행합니다.

### 기능

TextDownloader 노드는 사용자가 제공한 텍스트를 지정된 파일명으로 로컬에 저장합니다. 파일 형식은 기본적으로 `.txt`이며, 사용자는 파일명과 텍스트 내용을 입력으로 제공합니다. 이 노드를 사용하면 텍스트 데이터를 손쉽게 파일로 저장하여 보관할 수 있습니다.

### 언제 사용할까요?

TextDownloader 노드는 다음과 같은 상황에서 유용하게 사용됩니다:

- 프로세스 중 생성된 텍스트 데이터를 로컬 파일로 저장해야 할 때
- 로그, 보고서 또는 기타 텍스트 정보를 파일로 보관하고자 할 때
- 자동화된 워크플로우에서 텍스트 저장 단계를 포함해야 할 때

<Info>이 노드는 입력된 텍스트를 로컬 파일로 저장하여 데이터 관리와 보관을 간편하게 합니다.</Info>
