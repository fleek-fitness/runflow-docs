### 노드 입력

- `value` (string): 출력에 전달될 단일 문자열 값입니다. Node의 기본 입력 값으로 사용됩니다.

### 노드 출력

- `output` (string): 입력으로 제공된 `value`를 그대로 반환한 결과 값입니다.

### 기능

InputNode는 지정된 `value`를 입력으로 받아 그대로 출력하는 단순한 역할을 수행합니다. 사용자가 정의한 값을 전달하고 그 값을 다른 Node로 전송하기 위한 기본적인 Node입니다.

### 언제 사용할까요?

이 Node는 작업의 시작 단계에서 기본 값을 설정하거나 특정 값을 다른 Node로 전달할 필요가 있을 때 유용합니다. 간단한 값을 입력하여 워크플로우 내에서 일관되게 사용할 수 있도록 합니다.
