```plaintext
### 맞춤 포트

Mintlify는 기본적으로 포트 3000을 사용합니다. Mintlify가 실행되는 포트를 사용자 정의하려면 `--port` 플래그를 사용할 수 있습니다. 예를 들어, 포트 3333에서 실행하려면 다음 명령어를 사용하세요:
```
```bash
mintlify dev
```
같은 포트에서 이미 사용 중인 경우, Mintlify를 실행하려고 하면 다음과 같은 오류가 발생할 수 있습니다:
```bash
mintlify dev --port 3333
```
